# FLOWBLOK - AI Native Visual Business Operating System

## Mission

Build a single platform that combines the best parts of Storyblok, WordPress, Strapi, Contentful, Builder.io, Webflow, Shopify, WooCommerce, Zoho CRM, n8n, Boomi, Supabase, and modern AI tools.

The goal is to allow any non-technical user to build and manage websites, applications, ecommerce stores, portals, CRM systems, databases, workflows, APIs, and digital experiences visually without coding.

At the same time, developers must have complete freedom to access, customize, extend, and modify every layer of the platform.

This is not a CMS.

This is a Digital Experience Operating System.

---

# Core Philosophy

Most platforms separate everything:

Storyblok = Content

Strapi = Database

n8n = Workflows

Zoho = CRM

Shopify = Commerce

Builder.io = Visual Design

Vercel = Deployment

The user must learn many tools.

Flowblok combines everything into one platform.

Every object should be manageable from a single interface.

A page should not only have design settings.

A page should also manage:

Design

Content

Database

Logic

Permissions

SEO

Analytics

Caching

AI

API Connections

Workflows

Assets

Everything should be connected.
