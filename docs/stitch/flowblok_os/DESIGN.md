# Flowblok OS — Elite System Design

Design a world-class product experience inspired by Linear, Raycast, Vercel, Framer, Arc Browser, and modern Apple software.

The interface must feel: Fast, Quiet, Technical, Premium, Precise, Minimal.

## Colors
- **Canvas/Background**: #FFFFFF
- **Elevated Surfaces**: #FAFAFA
- **Borders**: #EEEEEE
- **Primary Text**: #111111
- **Secondary Text**: #666666
- **Accent**: #2563EB (Only one accent color)
- **Palette**: Monochrome with single blue accent. No gradients. No rainbows.

## Typography
Font: Inter (weights 400, 500, 600 only. Never 700+).
- **Display**: 64–80px, Tracking: -0.04em, Line Height: 0.95
- **Heading**: 32–48px, Weight: 600
- **Body**: 16px, Weight: 400, Line Height: 1.6
- **Labels**: 12–13px, Uppercase, Tracking: 0.08em

## Layout & Geometry
- **Spacing**: 8px system. 120px vertical spacing for major sections.
- **Max Width**: 1200px.
- **Radius**: Buttons 8px, Cards 8px.
- **Philosophy**: Asymmetrical layouts. Generous whitespace. Minimalist borders. Almost no shadows.

## Navigation
- **Height**: 56px.
- **Style**: Invisible feel. Minimal separators. No boxed menu items.
- **Active State**: Subtle (thin indicator or soft background).

## Components
- **Buttons**: 36-40px height, 8px radius, minimal shadows.
- **Inputs**: Subtle borders, quiet focus states.
- **Cards**: Border-first, shadows only when necessary.
- **Tables**: Lightweight, primary data surface.
- **Workflow Builder**: Professional engineering tool. Dark canvas (#0A0A0A), subtle grid, thin lines, elegant node cards, zero visual noise.
