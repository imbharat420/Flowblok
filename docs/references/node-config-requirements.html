3 · Tabs & actions inside the NDV

Parameters tab

  • All node params, top→bottom, with fx toggles
  • Add Option reveals optional/advanced params
  • Inline hints / placeholders / required markers

Settings tab

  • Node notes
  • Continue on fail (don't stop the run)
  • Retry on fail (attempts, wait)
  • Notes-in-flow, rename, execute-once

Execute step

  • Run the workflow up to this node
  • Populate INPUT (items received) + OUTPUT (items produced)
  • Show errors inline

INPUT / OUTPUT panes

  • JSON view of items in / out
  • Pin data for testing (n8n: "pinned data")
  • Item count + per-item navigation

4 · Node-add panel

The + / search opens a searchable node picker grouped by category (Triggers, Core, Flow, Data, AI, Integrations) — and for the AI Agent's sub-ports, a filtered picker (only Chat Models / Memories / Tools).
